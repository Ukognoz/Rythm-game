Rythmus Jumper - Reddit Open Beta

Thank you for testing the open beta.

How to start:
1. Install Java 8 or newer.
2. Unzip this folder.
3. Double-click Start-Rythmus-Open-Beta.bat.

Beta notes:
- The game starts in English by default.
- Login, leaderboard, cloud progress, and replay upload use Firebase.
- Replays are saved locally in the "replays" folder and uploaded when logged in.
- This public beta does not include the admin panel.
- Please report bugs with your account name, level, approximate time, and what happened.

Security note:
This public build removes debug symbols and hides backend strings from simple class-file inspection, but any client-side game can still be reverse engineered. Backend database rules remain the real protection.
